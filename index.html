IST 363 - Advanced Web Development

Welcome to IST 363!

This course will introduce students to emerging web technologies to create robust, high quality front-end web interfaces. Students will learn advanced page layout and styling techniques as well as responsive development and interactivity with JavaScript.

Classroom & Meeting Times

Hinds Hall, Room 243 (ICEBOX II), Tuesday and Thursday 5:00 - 6:20pm

Office Hours

Office hour will be available immediately after class for 30 mins.

Office hours will also be available Wednesday evenings from 7pm-9pm virtually by appointment.

Learning Objectives

After taking this course, the students will be able to:

  • Demonstrate the semantic meaning of HTML tags, HTML5 structural elements, ARIA roles and landmarks and how to better promote accessibility.
  • Apply the Flex Box layout module to create advanced page layouts that are responsive and cross browser and platform compatible.
  • Utilize advanced styling capabilities of CSS3 as they relate to typography, box properties, animations and transitions.
  • Utilize JavaScript to add interactivity to promote user engagement.
  • Experiment with JavaScript libraries and frameworks to handle events, initialize plugins, and interact with REST API's.

Book
